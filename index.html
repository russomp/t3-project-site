The project

The idea management tool is currently in beta. It is being worked on as part of CS577a at USC, in collaboration with System1 over the Fall 2018 semester.

The current product iteration attempts to streamline the idea management and content approval process by allowing CMS administrators to set and track content goals for their team in an easy to use webapp. It also allows content contributors to pitch ideas for admin approval, and monitor their personal and team progress more granularly.

The core web application was developed using python Flask for the backend API with a React component library on the frontend.
